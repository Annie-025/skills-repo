# 写作、翻译与整理

## `baoyu-translate`

- 来源：`JimLiu/baoyu-skills`
- 本地路径：`/Users/chenxinyu/.codex/skills/baoyu-translate`
- 功能：文章和文档翻译。
- 模式：quick、normal、refined。
- 适用场景：英中互译、精翻、本地化、文章翻译、长文分块翻译、术语一致性处理。
- 注意：首次使用时会要求创建 `EXTEND.md` 偏好，这是该 skill 的阻塞步骤。

## `baoyu-format-markdown`

- 来源：`JimLiu/baoyu-skills`
- 本地路径：`/Users/chenxinyu/.codex/skills/baoyu-format-markdown`
- 功能：Markdown / 纯文本整理。
- 能力：frontmatter、标题、摘要、层级标题、加粗、列表、代码块、中文排版修正。
- 适用场景：整理草稿、优化文章结构、统一 Markdown 格式。
- 注意：原则上只调整格式和明显错别字，不主动增删改写原内容。

## 建议

这组适合把原始材料变成可发布内容。推荐先为 `baoyu-translate` 设置个人默认偏好，例如默认翻译到 `zh-CN`、模式 `normal`、风格 `storytelling` 或 `technical`。
